# 💳 Sistema Bancário em Java

Projeto desenvolvido para praticar os conceitos fundamentais de Programação Orientada a Objetos (POO) utilizando Java.

O sistema simula um caixa eletrônico via terminal, permitindo o gerenciamento de contas bancárias e operações básicas como abertura de conta, depósitos, saques, consulta de saldo e encerramento da conta.

## 🚀 Funcionalidades

* Criar uma conta bancária
* Abrir conta
* Fechar conta (somente quando o saldo for igual a zero)
* Realizar depósitos
* Realizar saques com validação de saldo
* Consultar saldo da conta
* Gerenciar múltiplas contas utilizando `ArrayList`

## ▶️ Como executar

1. Clone este repositório.
2. Abra o projeto em sua IDE Java de preferência.
3. Execute a classe `Main`.
4. Utilize o menu do terminal para realizar as operações bancárias.
